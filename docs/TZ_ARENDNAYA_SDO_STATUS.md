# Статус выполнения ТЗ «Арендная СДО»

Трекер прогресса по [TZ_TRUDSKILL_ARENDNAYA_SDO.md](../TZ_TRUDSKILL_ARENDNAYA_SDO.md) (дельта-ТЗ к `SDOPROF_TZ_FINAL.md`). Агент по фразе **«продолжай по ТЗ»** читает этот файл, чтобы понять: **какая фаза текущая, что сделано, что частично, что не начато и что нужно переделать**.

## Правила ведения

1. Обновляется **в конце каждой сессии**, работавшей по этому ТЗ, — вместе с handoff `§5.XX` и README `§2` (см. [DOCUMENTATION_MAP.md — протокол передачи](./DOCUMENTATION_MAP.md#agent-handoff-protocol)).
2. Статусы: **✅ готово** (критерий приёмки выполнен) · **🔄 в работе** · **🟡 частично** (фундамент есть, довести по ТЗ) · **⬜ не начато** · **♻️ переделать** (сделано, но не так, как требует ТЗ).
3. Текст требований сюда **не копировать** — только номер ФТ, статус, короткий указатель на код/handoff. Канон текста — само ТЗ.
4. Фаза стартует только после **плана в `docs/superpowers/plans/` и апрува владельца** (ТЗ §13, §15). Начальные статусы ниже — из аудита в самом ТЗ (коммит `ac2af17`) + факта кода на 2026-07-23.
